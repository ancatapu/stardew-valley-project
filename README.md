Final project for the Aide en Ligne course.

# About the Stardew Valley project
This repository contains the documentation for the Stardew Valley final project. It provides a simple guide for new players to make the most out of the game during their first week on the farm.
It contains 3 files :
* __concept topic:__ describes the basics of stardew valley
* __task topic:__ indicates how to plant crops
* __reference topic:__ references all crops per season in game

All files have been imported in the second branch (anca-stardew), then commited to the main branch through a pull request.
